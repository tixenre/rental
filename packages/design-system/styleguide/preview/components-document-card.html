Tipos de documento: cotización (azul) → contrato (verde) → remito de entrega (amber) → albarán de devolución (neutro). La disponibilidad de cada documento sigue el estado del pedido.

Estados: disponible (hover con descarga), generando (progress bar animado), pendiente (opacidad 45%, sin hover), error (destructive ring + retry icon).

API: GET /api/pedidos/:id/documentos devuelve un array con tipo, url, generado_at, estado. La descarga es un link directo a la URL firmada de S3 (o similar).
