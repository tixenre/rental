Páginas completas: las error pages usan font-display (Champ Black) para el código de error en lowercase — es uno de los pocos casos en que el display font aparece fuera del hero, para darle carácter. El copy es empático y accionable.

404: CTA primario siempre al catálogo. 500: CTA a reintentar + ir al inicio. Sin red: solo reintentar.

Inline errors: usan el componente Banner (ver components-banners.html) como base, con un ícono contextual y un link de acción. No usar toast para errores persistentes — usar banner inline o estado de error en el mismo lugar donde ocurrió.
