Generación PDF: los documentos (cotización, contrato, remito, albarán) se generan server-side con @react-pdf/renderer o puppeteer en el backend FastAPI. El frontend no genera PDFs — consume la URL firmada del archivo pre-generado.

Print CSS: la hoja src/styles/print.css se importa globalmente. Oculta toda la UI de navegación. El documento HTML de la cotización tiene clase .printable-doc para targeting específico.

Tipografía en print: mínimo 12pt. Usar font-sans (TT Commons) que está embebida. font-mono (JetBrains) para precios y números. Champ Black para el wordmark del header.
