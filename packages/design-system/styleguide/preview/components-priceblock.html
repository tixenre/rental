Nombre en TT Commons · precio SIEMPRE font-mono · nunca Champ Black
// PriceBlock — props
<PriceBlock
  perDay={24500}          // número crudo ARS
  jornadas={3}            // multiplica si conTotal
  qty={1}
  conIva={true}          // muestra "neto + IVA", no lo suma
  size="md" />            // sm · md · lg

formatARS(24500)  // → "$ 24.500"  (punto miles, espacio post-$)
priceBreakdown(items, { conIva: true })  // → { neto, iva } · se muestra "neto + IVA"
Regla de oro: precios SIEMPRE en font-mono con tabular-nums — nunca Champ Black, nunca font-sans. · formatARS() es la única fuente de formato: $ + espacio + punto como separador de miles, sin decimales salvo IVA. · Si perDay == null → "consultar" en muted. · Con IVA se muestra como neto + IVA (ej. $ 145.500 + IVA), nunca el IVA ya sumado al total — solo en carrito/checkout, no en cards de catálogo.
