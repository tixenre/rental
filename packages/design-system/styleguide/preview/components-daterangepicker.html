Lógica de estados
/* Bloqueo — impide confirmar */
const blocked = devolucionCerrada || rangoCruzaBloqueado;

/* Jornada extra — avisa, no bloquea */
const sumaJornadaPorHora = timeToMinutes(endTime) > timeToMinutes(startTime);

const canApply = hasStart && !blocked;

/* Jerarquía de feedback — una sola bar por vez */
{devolucionCerrada   && <Bar variant="error">  La devolución cae el {día}, cerrado   </>}
{rangoCruzaBloqueado && <Bar variant="error">  El {fecha} no hay stock para…         </>}
{sumaJornadaPorHora  && <Bar variant="amber">  Devolvés a las {hora}, suma 1 J      </>}
{!hasStart           && <Bar variant="muted">  Tocá un día en el calendario…        </>}

/* Devolución card — color según estado */
devolucionCerrada  → border-destructive/40  bg-destructive/10   fecha en text-destructive
sumaJornadaPorHora → border-naranja/45      bg-naranja/12       badge +1 J
default            → border-amber/45        bg-amber/14
Jornada = día completo de retiro. La devolución se hace al inicio del día retiro + N jornadas — el cálculo es puramente de días, no de horas, salvo el caso de "jornada extra por hora" que es una advertencia al usuario. · Días cerrados vienen de GET /api/settings/closed_days (domingos + feriados configurables desde admin). · Sin stock se consulta por ítem: si el carrito tiene múltiples ítems, se evalúa el más restrictivo.
