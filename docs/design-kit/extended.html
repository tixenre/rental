09 · Voz y tono

Cómo habla Rambla

Tono cercano, directo, rioplatense. "Vos", minúsculas en el copy de marca, puntuación que da ritmo.

Minúsculas en marca
El wordmark, los taglines y los títulos de hero van en lowercase. Headings de UI van en Title Case normal.
rental, estudio, rambla.
Tuteo rioplatense
Siempre "vos" y segunda persona singular. No "usted", no "tú". El tono es el de un colega del rubro.
reservá · elegí · confirmá
Punto final como remate
Los taglines y titulares de hero terminan con punto. No signo de exclamación, no elipsis. El punto es autoridad, no frialdad.
en rambla, tu rodaje.
Números sin ambigüedad
Precios con $ sin espacio, separador de miles con punto. Días con "J" abreviado. Rangos con "→".
$ 24.500 · 3 J · lun → jue

19 · Imagen & fotografía

Aspect ratios y estilo visual

Los equipos siempre en 1:1. El hero de landing acepta 16:9 o full-bleed. Fondos de card en amber-soft cuando no hay foto.

1 : 1
Equipment card Foto del equipo. Fondo blanco o amber-soft si no hay foto. Nunca crop.
16 : 9
Hero / banner Landing, sección "Conocé el Estudio". Puede ser full-bleed.
4 : 5
Perfil / testimonio Fotos de personas, testimonios de clientes.
3 : 1
Banner angosto Categorías, strips promocionales.
Reglas de fotografía
Estilo fotográfico
Fondo claro o neutro, iluminación limpia. Sin filtros saturados. El equipo como protagonista, no en contexto.
Alt text
Describir marca + modelo + tipo: "Cámara Sony A7S III, vista frontal". No "imagen de equipo".
Fallback sin foto
Fondo amber-soft con las iniciales del modelo en Champ Black, 25% opacity ink.
Formato y tamaño
WebP preferido. Cards: 600×600 @2x. Hero: 1920×1080 mínimo. Lazy load siempre.

Amber scroll — topbar que cambia al hacer scroll
/* kit/components/topbar.tsx */
import { useEffect, useState } from "react";

export function TopBar() {
  const [scrolled, setScrolled] = useState(false);

  useEffect(() => {
    const onScroll = () => setScrolled(window.scrollY > 48);
    window.addEventListener("scroll", onScroll, { passive: true });
    return () => window.removeEventListener("scroll", onScroll);
  }, []);

  return (
    <header className={cn(
      "sticky top-0 z-[var(--z-sticky)] transition-colors",
      "duration-[var(--duration-base)] ease-[var(--ease-out)]",
      scrolled
        ? "bg-amber border-b border-amber"     // amber al scrollear
        : "bg-background/95 border-b border-hairline backdrop-blur-sm"
    )}>
      ...
    </header>
  );
}
