Implementación
/* print.css — importado solo cuando se monta una vista imprimible */
@media print {
  @page { size: A4; margin: 16mm; }

  /* Ocultá TODO el chrome de la app */
  .topbar, .cart-minibar, nav, button, .toaster { display: none !important; }

  /* Colores planos — sin oklch, sin tokens de pantalla.
     La impresora no entiende --background hueso; usá tinta directa. */
  body { background: #fff; color: #14100c; }

  /* Champ Black SOLO en el wordmark; el resto TT Commons */
  .lh-mark { font-family: "Champ Black"; }

  /* Precios SIEMPRE tabular — alinean en columna */
  .num { font-variant-numeric: tabular-nums; text-align: right; }

  /* Evitá cortar filas entre páginas */
  tr, .totals { break-inside: avoid; }
}
Tinta plana, no tokens de pantalla: en print pasá a hex directos (#14100c ink, #fff papel, #FFF6E0 banda amber). Los oklch del hueso se ven sucios impresos. · Champ Black solo en el wordmark — el resto del documento es TT Commons, igual que en pantalla. · Precios tabular-nums + alineados a la derecha para que las columnas cuadren. Siempre por formatARS(). · Período en formato de marca: lun 2 → jue 5 jun. + jornadas. · break-inside: avoid en filas y totales para que no se corten entre hojas. · Este pattern cubre presupuesto (válido N días) y remito (cambia el badge y saca totales/IVA).
