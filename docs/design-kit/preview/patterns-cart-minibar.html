Contrato de data-attributes
API entre CartMiniBar y FlyToCartLayer
data-cart-icon Marcador en el círculo del ícono del carrito. El FlyToCartLayer hace querySelector('[data-cart-icon]') para leer el getBoundingClientRect() como punto de aterrizaje de la animación "+1". Si no está en el DOM, la animación no dispara.
data-no-print Agregar al .mini-bar para que no aparezca en impresión/PDF del pedido.
Tokens CSS clave
/* Borde top signature — separador del carrito */
.mini-bar {
  border-top: 2px solid color-mix(in oklch, var(--amber) 60%, transparent);

  /* Sombra que sube (no default) */
  box-shadow: 0 -8px 32px -8px rgba(0,0,0,0.15);

  /* Casi sólido + blur — legible sobre imagen */
  background: color-mix(in oklch, var(--background) 98%, transparent);
  backdrop-filter: blur(20px);

  /* Safe area iOS — home indicator */
  padding-bottom: max(0.75rem, env(safe-area-inset-bottom));
}

/* Hover preview — slide-up (CSS, sin JS) */
.hover-preview {
  transform: translateY(8px); opacity: 0;
  transition: opacity 150ms var(--ease-out),
              transform 150ms var(--ease-out);
  pointer-events: none;
}
/* Solo visible en desktop con puntero real */
@media (hover: hover) and (min-width: 1024px) {
  .mini-bar:hover + .hover-preview,
  .hover-preview:hover {
    transform: translateY(0); opacity: 1;
    pointer-events: auto;
  }
}
Tailwind class exacta: fixed inset-x-0 bottom-0 z-40 border-t-2 border-amber/60 bg-background/98 shadow-[0_-8px_32px_-8px_rgba(0,0,0,0.15)] backdrop-blur-xl · Hover preview: usa group/cart + group-hover/cart: (Tailwind named groups). Transición: translate-y-2 opacity-0 → translate-y-0 opacity-100 en 150ms. Solo lg:[@media(hover:hover)] · Icon animation: motion.div con data-cart-icon.
