Contrato de data-attributes
API entre EquipmentCard y FlyToCartLayer
data-cart-icon En el círculo del carrito (CartMiniBar). El FlyToCartLayer hace querySelector('[data-cart-icon]') para leer getBoundingClientRect().center como destino del vuelo. Si no está en el DOM → no anima.
flyKey (store) Timestamp que actúa como key del AnimatePresence. Permite re-disparar la animación aunque el origin sea el mismo (taps rápidos = la última reemplaza a la anterior, nunca se acumulan).
Implementación (vanilla — sin Framer Motion)
/* Curva signature del vuelo — siempre esta, no otra */
const EASING = "cubic-bezier(0.22, 1, 0.36, 1)";
const DURATION = 550;   // ms — vuelo total
const POP_DELAY = 500;  // ms — pop del ícono al llegar

function flyToCart(originEl) {
  const origin = originEl.getBoundingClientRect();
  const target = document.querySelector("[data-cart-icon]")
                   ?.getBoundingClientRect();

  if (!target) return; // barra no montada → no animar

  /* Crear pill */
  const pill = document.createElement("div");
  pill.textContent = "+1";
  pill.className = "fly-pill";
  pill.style.cssText = `
    left: ${origin.left + origin.width / 2 - 18}px;
    top:  ${origin.top  + origin.height/ 2 - 18}px;
    pointer-events: none; aria-hidden: true;
  `;
  document.body.appendChild(pill);

  /* Calcular desplazamiento final */
  const dx = (target.left + target.width / 2) - (origin.left + origin.width / 2);
  const dy = (target.top  + target.height/ 2) - (origin.top  + origin.height/ 2);

  /* Web Animations API — curva en dos etapas */
  pill.animate([
    { transform: "translate(0,0) scale(1)",     opacity: 1 },
    { transform: `translate(${dx}px,${dy}px) scale(0.4)`, opacity: 0.8 },
  ], { duration: DURATION, easing: EASING, fill: "forwards" })
     .onfinish = () => pill.remove();

  /* Pop del ícono destino al llegar */
  setTimeout(() => {
    document.querySelector("[data-cart-icon]")
      ?.classList.add("pop");
    setTimeout(() =>
      document.querySelector("[data-cart-icon]")
        ?.classList.remove("pop"), 450);
  }, POP_DELAY);
}
z-index: 60 (z-[60] en Tailwind) — encima del CartMiniBar (z-40) y el TopBar (z-50 via --z-topbar). · Pill exacto: h-9 w-9 (36px) · -translate-x-1/2 -translate-y-1/2 centrado en coords del store · ring-2 ring-amber/40 halo. · onExitComplete: el pop del ícono se dispara en AnimatePresence onExitComplete, no en onAnimationComplete. · flyKey: counter incremental en Zustand — triggerFly() incrementa el counter que es la key del AnimatePresence. Taps rápidos reemplazan sin apilar.
