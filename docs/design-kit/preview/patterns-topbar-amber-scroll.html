Topbar amber transition · driven by scroll progress (0 → 1)
0%
The continuous part: --amber-pct CSS variable drives the background of the topbar (color-mix(in oklch, var(--amber) var(--amber-pct), bone)).
The snap part: when progress crosses 65%, the topbar gets .scrolled class. Logo SVG fill jumps amber→bone, date pill flips to bone-bg/ink-text, user button flips to bone-bg/ink-icon.
Triggering: in JSX use a scroll listener on the hero ref. Calculate progress as 1 - clamp((heroBottom - 64) / (heroHeight * 0.5)). Set --amber-pct on the topbar and toggle .scrolled at progress > 0.65.
